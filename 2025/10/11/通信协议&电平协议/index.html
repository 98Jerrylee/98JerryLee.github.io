通信接口&电平标准

一、通信接口 / 总线

这些定义帧格式、握手、链路管理；具体电平依赖下一节的物理层实现。

1.1 UART/USART（常配 TTL/RS-232/RS-485 物理层）

1.2 SPI

1.3 I²C

1.4 SDIO

1.5 eMMC

1.6 LVDS

1.7 JESD204B

1.8 USB 2.0/3.x

1.9 Ethernet 10/100/1G/10G/25G

1.10 PCI Express Gen1–5

1.11 Aurora（Xilinx 轻量级高速串行）

二、物理层 / 电平标准

这些定义电压摆幅、差分/单端、终端方式、速率能力；上层可承载多种协议。

2.1 单端逻辑电平

2.1.1 TTL / LVTTL

2.1.2 LVCMOS（3.3V / 2.5V / 1.8V / 1.5V / 1.2V）

2.1.3 SSTL-I/II（1.5V/1.8V，DDR 存储接口）

2.1.4 HSTL-I/II（1.5V/1.8V，高速单端）

2.2 差分电平

2.2.1 RS-422 / RS-485

2.2.2 LVDS

2.2.3 CML（高速串行、SerDes、PCIe/DP）
